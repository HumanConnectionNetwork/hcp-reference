# HCP Reference

The official reference implementation of the Humanitarian Connection Protocol (HCP).

This project provides the core services required to create, validate, manage and exchange standardized humanitarian records according to the HCP specification.

It serves as the canonical implementation of the protocol and can be used as a foundation for humanitarian platforms, government systems, NGOs and HCP-compatible applications.

The project itself is independent of any specific client or user interface. Applications such as Telegram bots, mobile apps or web platforms interact with this implementation through the HCP interfaces.
