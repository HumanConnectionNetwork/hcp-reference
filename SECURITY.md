# Security Policy

## Reporting a Vulnerability

The Humanitarian Connection Protocol (HCP) is intended to support humanitarian systems where data integrity and trust are critical.

If you discover a security vulnerability, please do not open a public GitHub issue.

Instead, report it privately by contacting:

security@humanconnectionnetwork.org

(or the appropriate contact once available)

Please include:

- Description
- Impact
- Steps to reproduce
- Suggested mitigation (if known)

We will acknowledge your report as soon as possible and work with you toward a responsible disclosure.

---

## Responsible Disclosure

Please allow reasonable time for vulnerabilities to be investigated and resolved before public disclosure.

---

## Scope

Security reports may include issues affecting:

- Protocol implementation
- Identity validation
- Record integrity
- Authentication
- Authorization
- APIs
- Dependencies

---

Thank you for helping improve the security of humanitarian infrastructure.
